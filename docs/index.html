Why it exists

Built for the hard cases

Most tools assume a clean, well-formed APK and one decompiler. Malware and protected apps are not like that. That gap is the whole reason Mythosaur exists.

Opens APKs others won't

Hardened packages ship a deliberately broken archive that strict tools reject, even though Android installs them fine. Mythosaur repairs it and analyses it anyway.

Four decompilers, auto-fallback

Every class can be rebuilt four ways, scored for quality, best picked for you. When one chokes on obfuscated control flow, another usually reconstructs it cleanly.

Beats string encryption

A sandboxed Dalvik interpreter runs one method — say a string-decryptor — and shows its real return value plus a register trace. No device, and you never run the app.

Bytecode when source fails

Even if a class never decompiles, you still get its methods, strings, cross-refs, control-flow graph and editable smali straight from the DEX.

In the box

Everything in one window

Read, analyse, patch and re-sign without stitching command-line tools together by hand.

App navigation flow

A top-down map of which Activity opens which, so you can see how the app moves screen to screen — something most tools don't draw at all.

Method control-flow graph

Each method's basic blocks drawn as a graph, with branch and loop edges — the function-internals view you'd expect from Cutter.

Deep DEX analysis

Filterable tables of classes, methods, imports and strings, plus cross-references and a clickable call graph for any method.

Protections & permissions

Packer, obfuscation and anti-RE detection, an entropy scan for hidden payloads, and a map linking each permission to the code that actually uses it.

Patch & re-sign

Edit smali for code, or full-decode for resources and the manifest, then rebuild, zipalign, sign and verify (v1/v2/v3), and install over adb.

Native & debugging

ELF inspection with ARM64/ARM/x86 disassembly, plus an optional live smali debugger (breakpoints, stepping, registers) on a debuggable device.

  • Multi-engine code view with syntax highlighting and in-editor find (Ctrl+F)
  • Live-filtered class tree that cuts a 40k-class app down to size
  • Global search across names, strings, or full decompiled source
  • Tolerant manifest parsing for mangled binary manifests
  • Reusable workspaces that restore your edits and patched APKs
  • Live dark / light theme

Download .exe (v1.0.0) 
# Linux — install and launch
sudo apt install ./mythosaur-re_1.0.0_amd64.deb
mythosaur-re

# or build & run from source (needs JDK 21+)
git clone https://github.com/Durgeshkt/mythosaur-re.git
cd mythosaur-re && ./gradlew run

Optional for the full pipeline: zipalign + apksigner (patching), adb (install to device), and a disassembler for native .so — binutils-multiarch, llvm, or per-arch binutils-<arch>.
